# JA Nekretnine

Prezentacijski web sajt za agenciju za promet nekretnina **JA Nekretnine**.

## O sajtu

Jednostranični (one-page) sajt namijenjen prezentaciji usluga ekskluzivne agencije za nekretnine. Naglasak je na luksuznom dizajnu, povjerenju i osobnom integritetu agenta – što i sam slogan „Ja sam ime i garancija" naglašava.

## Sadržaj

- **Hero sekcija** – uvodna slika s naslovom i pozivom na akciju
- **O nama** – opis agencije i filozofije poslovanja
- **Šta radimo drugačije** – četiri ključne prednosti (strateško pozicioniranje, pregovaranje, zaštita klijenta, end-to-end vođenje)
- **Usluge** – ekskluzivna prodaja, kupovina, iznajmljivanje i poslovne nekretnine
- **Nekretnine** – prikaz dostupnih oglasa
- **Filozofija** – vrijednosti agencije (diskrecija, preciznost, povjerenje, integritet)
- **Kontakt forma** – slanje poruke putem PHP Mailera

## Tehnologije

- HTML5, CSS3, JavaScript (vanilla)
- PHP + PHPMailer (kontakt forma)
- Google Fonts (Playfair Display, Inter)

## Jezik

Bosanski (bs)
