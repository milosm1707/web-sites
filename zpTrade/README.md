# ZP Trade

Prezentacijski web sajt za poljoprivrednu kompaniju **ZP Trade** – pouzdanog partnera u poljoprivredi.

## O sajtu

Višestranični sajt koji predstavlja asortiman poljoprivrednih proizvoda i usluga koje nudi ZP Trade. Dizajn koristi zelenu i zlatnu paletu boja, prilagođenu poljoprivrednoj tematici. Sajt sadrži više stranica za različite kategorije ponude.

## Stranice

| Fajl | Opis |
|------|------|
| `index.html` | Naslovna – pregled firme i ključnih kategorija proizvoda |
| `products.html` | Katalog proizvoda |
| `about.html` | O kompaniji |
| `contact.html` | Kontakt stranica |

## Kategorije proizvoda

- 🌱 Gnojiva i đubriva
- 🌾 Sjeme i sadni materijal
- 🐛 Pesticidi i zaštitna sredstva
- 💧 Oprema za navodnjavanje
- 🐄 Stočna hrana
- 🚜 Poljoprivredna mehanizacija

## Tehnologije
