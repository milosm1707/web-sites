# Spitex PflegeAufRädern GmbH

Prezentacijski web sajt za **ambulantnu njegu na kućnoj adresi** – Spitex PflegeAufRädern GmbH iz Wangena pri Dübensdorfu (Švicarska).

## O sajtu

Profesionalni sajt namijenjen predstavljanju medicinskih i njegu pružajućih usluga u kantonu Zürich. Optimizovan za lokalno pretraživanje (SEO) i prilagođen svim uređajima. Sadrži Schema.org oznake tipa `MedicalBusiness` te Open Graph tagove za dijeljenje na društvenim mrežama.

## Sadržaj

- **Hero sekcija** – poziv na zakazivanje termina, ključne statistike (24h dostupnost, 100% certificirano)
- **Über uns (O nama)** – predstavljanje tima i pristupa njezi
- **Dienstleistungen (Usluge)** – medicinska njega, lična briga, kućni poslovi (Hauswirtschaft)
- **Kontakt** – telefon, e-mail i kontakt forma (PHP Mailer)
- **24/7 Pikettdienst** – istaknutost dežurne službe

## Tehnologije

- HTML5, CSS3, JavaScript (vanilla)
- PHP + PHPMailer (kontakt forma)
- Schema.org – MedicalBusiness structured data
- Google Fonts
