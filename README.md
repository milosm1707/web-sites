## Tehnologije

Svi sajtovi su izrađeni koristeći:

- **HTML5, CSS3, JavaScript** (vanilla, bez frameworka)
- **PHP + PHPMailer** (kontakt forme tamo gdje je potrebno)
- **Google Fonts** (prilagođene tipografije po projektu)
- **Schema.org** structured data (SEO, tamo gdje je relevantno)
